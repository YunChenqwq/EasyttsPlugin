# 安装指南（给小白）

## 方式 1：MaiBot WebUI 插件市场安装（推荐）

1) 打开 MaiBot WebUI → **插件市场**  
2) 搜索关键词：`tts`  
3) 找到本插件：**EasyPlugin / yunchenqwq.easy_plugin**  
4) 点击安装 → 等待安装完成  
5) 重启 MaiBot（或在 WebUI 里重载插件）

> 安装完成后，在 WebUI 的“已安装插件”里也能看到本插件。

---

## 方式 2：手动安装（适合离线/无法访问插件市场）

1) 下载/克隆仓库：`https://github.com/YunChenqwq/EasyttsPlugin`  
2) 把插件目录放到你的 MaiBot 插件目录，例如：

```
E:\bot\MaiBotOneKey\modules\MaiBot\plugins\EasyPlugin\
```

3) 重启 MaiBot（或在 WebUI 里重载插件）

> 如果依赖没有自动装上：在 MaiBot 的 Python 环境里执行 `pip install -r requirements.txt`。
